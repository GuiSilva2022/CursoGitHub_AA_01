1) O que é um repositório?
  Repositório é lugar para armazenar pacotes de software que podem ser recuperados e instalados em seu computador
2) Qual a diferença entre Git e Github?
  Git é um software local que permite aos desenvolvedores salvar seus projetos ao longo do tempo. GitHub é uma plataforma na web que junta os recursos de controle de versões do gitHub.    
3) Porque utilizar controle de versão para os arquivos de um projeto de desenvolvimento de software?
  É importante usar o controle de versão porque ele ajuda os desenvolvedores a se "mover" mais rápido e permite que as equipes de software preservem a agilidade e a eficiência.
4) Quais vantagens você enxerga no Git? Você utilizaria em seu projeto?
  As vantagens que eu enxergo é que é fácil de trabalhar e ele verifique o status de suas modificações.
